# seedance-skill

一个面向 Seedance / 即梦 2.0 场景的中文视频提示词 Skill。

该项目用于把用户的创意需求、图片/视频/音频参考素材、风格偏好和时长约束，整理为可直接复制使用的中文视频提示词。内容重点覆盖短剧、广告、电商展示、动作场面、音乐卡点、视频延长与编辑、多模态参考等常见生成任务。

## 项目特点

- 面向 Seedance / 即梦 2.0 的中文提示词设计
- 支持图片、视频、音频混合参考素材编排
- 支持首帧、尾帧、人物、场景、运镜、节奏、音色等用途标注
- 内置短视频、多版本输出、长视频分段延长等模式
- 根据题材自动切换风格抓手，如极道黑色电影、豪门短剧、战争大片、电商广告等
- 默认附带简洁且贴合镜头的负向提示词

## 适用场景

适合以下需求：

- Seedance / 即梦视频提示词生成
- 图生视频、视频生视频、带音频参考的视频生成
- 广告片、电商产品展示、品牌短视频
- 豪门恩怨、真假千金、打脸反转等短剧内容
- 黑色电影、极道对峙、动作追逐、战争逃生等强风格片段
- 一镜到底、运镜复刻、音乐卡点、视频延长与视频编辑

## 核心能力

- 将模糊创意整理成结构化中文提示词
- 为 `@图片N` / `@视频N` / `@音频N` 明确分配用途
- 根据时长选择简单模式、完整模式或超长模式
- 输出可直接复制的主提示词、负向提示词与可拼接版
- 在题材明确时直接给定稿式提示词，在方向不唯一时提供多个风格版本

## 文件说明

### `SKILL.md`
项目主文件，包含：

- Skill 元数据
- 适用范围与触发词
- 素材引用规则
- 提示词构造方法
- 输出模式定义
- 负向提示词与限制句策略
- 多类视频任务模板
- 风格映射与自检规则

### `references/`
风格参考资料目录，用于在特定题材下增强提示词质量：

- `japanese-yakuza-noir.md`：日本极道 / 新黑色电影
- `hollywood-war-blockbuster.md`：好莱坞战争大片
- `chinese-modern-shortdrama.md`：国产现代爽感短剧

## 使用方式

1. 阅读 `SKILL.md`，了解该 Skill 的适用范围与输出格式。
2. 在你的提示词工作流、Agent、Droid 或其他编排系统中加载 `SKILL.md` 作为技能定义。
3. 当用户提供题材、时长、比例与参考素材后，按规则输出中文视频提示词。
4. 若需求明显落入某种风格，可结合 `references/` 中对应模板强化镜头语言和叙事结构。

## 输出模式

项目内置三种主要输出模式：

- 简单模式：适合信息完整、只需要 1 条成品提示词的场景
- 完整模式：适合需要 2-3 个创意方向对比的场景
- 超长模式：适合超过 15 秒的视频，按“首段生成 + 后续延长”拆分输出

## 适合放到 GitHub 的原因

这个仓库适合作为：

- 面向短剧、广告和多模态视频生成的提示词资产库
- 个人或团队的视频生成提示词基线配置
